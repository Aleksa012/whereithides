# [Where It Hides]

## 🚀 Overview

This game is all about users creating simple but fun levels for other treasure-hunting lovers on Reddit. Users will be able to use simple mechanics like placing rocks, trees, and tools to create a straightforward, short, and fun experience and compete with fellow redditors on the leaderboard. May the fastest treasure hunter win!

- **Target Audience:** Casual gamers of all ages
- **Operational Notes:** Does not collect users' personal data except for score tracking for the leaderboard, where it saves the userId, username, score, and rank. Also, this same information is collected to stop users from farming points on the same level. Once passed, there are no repeats of points!

## 🛠️ How to Setup and Play

### For Moderators (Configuration)

1. Install the app via the Reddit App Directory.
2. For easier user navigation, pin the installed app so the create level option is easily accessible, as it doesn't appear on user-generated posts (levels).

### For Players (How to Play)

- **Controls:** Click on the map to move your character. Walk over tools (pickaxe/shovel) to collect them. A pickaxe allows you to remove an adjacent rock by tapping on it, making you able to walk on that square. The shovel digs the ground when you tap on the square you are currently standing on. On one square per level (if configured by the level poster), a map can be found revealing the winning square. Trees cannot be removed in any way.

Creating a Level: Select the type of tile you want to place via the available buttons containing icons of the tiles:

- **Rock tile** - rocks that can be broken if a pickaxe is collected
- **Tree tile** - an unbreakable obstacle serving a decorative purpose
- **Pickaxe** - the character needs it for breaking rock tiles
- **Shovel** - the character needs it for digging tiles
- **Map** - reveals the winning square to the player if found by digging
- **Start square (knight)** - marks the starting point for the level
- **Winning square (flag)** - marks the winning square for the level
- **Base tile (orange ground)** - resets the tile to its initial state

Notes:

- There can only be one winning square and one starting square (and not on the same tile).
- For a level to publish, it must be solvable; you can't submit an impossible level.
- Shovels, pickaxes, and maps can be placed under the rocks (base tile -> place pickaxe/shovel/map -> place rock). The player will be able to pick up the item beneath after breaking the rock.

* **Objective:** Win by standing on a winning square set by the level poster.

* **Sharing Scores:** Your score will be calculated based on the amount of clicks you needed to solve the level, so be mindful: more clicks = fewer points. Results will be automatically saved and can be accessed on the leaderboard, which shows you (the player) and up to 100 other players.
